## Getting Started
Lets get started! 

### Prerequisites

    npm install npm@latest -g

### Instalation
1.  Clone the repo
   
        git clone https://github.com/CaseyOverton/Portfolio-CO.git

2. Install NPM packages
   
        npm install
  
  3. Start using npm

            npm start
